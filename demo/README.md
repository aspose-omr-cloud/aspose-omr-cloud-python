# OMR Demo

## Requirements

* You need to complete [installation](https://github.com/aspose-omr-cloud/aspose-omr-cloud-demo-data#installation) procedure
* install `aspose-omr-cloud`

```sh
pip install aspose-omr-cloud
```

## Running OMR Demo

To run OMR Demo just execute

```sh
python run_demo.py
```
